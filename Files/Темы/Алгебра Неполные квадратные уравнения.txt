Неполные квадратные уравнения имеют  2  вида:
1. если  c = 0 , то  ax² + bx = 0 ;
	x⋅(ax+b) = 0 .
	x = 0   или  ax+b = 0
2. если  b = 0 , то  ax² + c = 0 .
  


ax² +c = 0  можно решить, извлекая корень из каждой части уравнения.
ax² = −c ; (обе стороны делятся на  a )  x² = −c/a .
  |x|=  √-c/a .   Извлекая корень из правой части уравнения, получаем  x  по модулю.
Это значит, что
x1   =  √-c/a ;
x2   =  -√-c/a.
